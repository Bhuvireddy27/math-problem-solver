Introduction to Probability Distributions

A probability distribution is a statistical function that describes the likelihood of obtaining the possible values that a random variable can take. These distributions are classified into two main types:

  • Discrete Probability Distributions: Used when the set of possible outcomes is finite or countable.
  • Continuous Probability Distributions: Used when the set of possible outcomes is infinite and uncountable (e.g., real numbers).

Discrete Probability Distributions

Discrete probability distributions describe situations where the random variable can take on a finite or countable number of values. Below are some common discrete distributions:

1. Binomial Distribution

The binomial distribution describes the number of successes in a fixed number of independent Bernoulli trials. Each trial has two outcomes: success or failure. The probability mass function (PMF) for the binomial distribution is:

P(X = k) = C(n, k) * p^k * (1 - p)^(n - k)

Where:

  • C(n, k) is the combination of n items taken k at a time,
  • p is the probability of success on a single trial,
  • n is the number of trials,
  • k is the number of successes.

Example: Coin Toss

If a coin is tossed 5 times (n = 5), the probability of getting exactly 3 heads (k = 3) is:

P(X = 3) = C(5, 3) * 0.5^3 * 0.5^2 = 10 * 0.125 * 0.25 = 0.3125

2. Poisson Distribution

The Poisson distribution is used for counting the number of events that occur in a fixed interval of time or space, given the average rate of occurrence. The probability mass function for the Poisson distribution is:

P(X = k) = (λ^k * e^(-λ)) / k!

Where:

  • λ is the average number of occurrences in the interval.
  • k is the actual number of occurrences of the event.
  • e is Euler's number, approximately 2.71828.

Example: Phone Calls

If a call center receives 4 calls per minute (λ = 4), the probability of receiving exactly 2 calls in a minute is:

P(X = 2) = (4^2 * e^(-4)) / 2! ≈ 0.1465

Continuous Probability Distributions

Continuous probability distributions describe situations where the random variable can take an infinite number of values within a given range. Common continuous distributions include:

1. Normal Distribution

The normal distribution, also known as the Gaussian distribution, is one of the most widely used continuous distributions. It is symmetric and follows a bell-shaped curve. The probability density function (PDF) for the normal distribution is:

f(x) = (1 / (σ√(2π))) * e^(- (x - μ)^2 / 2σ^2)

Where:

  • μ is the mean of the distribution,
  • σ is the standard deviation,
  • e is Euler's number (approximately 2.71828).

Example: Human Height Distribution

Suppose the heights of individuals in a population follow a normal distribution with a mean of 170 cm and a standard deviation of 10 cm. You can calculate the probability that a randomly chosen individual is taller than 180 cm using the Z-score formula.

2. Exponential Distribution

The exponential distribution is used to model the time between events in a Poisson process. It describes the time between successive events that occur at a constant average rate. The probability density function (PDF) for the exponential distribution is:

f(x) = λ * e^(-λx), for x ≥ 0

Where:

  • λ is the rate parameter,
  • x is the time between events.

Example: Time Between Calls

If the average time between customer service calls is 2 minutes (λ = 0.5), the probability of the time between two calls being greater than 3 minutes is:

P(X > 3) = e^(-0.5 * 3) ≈ 0.2231

3. Uniform Distribution

The uniform distribution is used when all outcomes are equally likely. The probability density function (PDF) for the uniform distribution is:

f(x) = 1 / (b - a), for a ≤ x ≤ b

Where:

  • a is the lower bound of the distribution,
  • b is the upper bound of the distribution.

Example: Fair Die Roll

The probability of rolling a 4 on a fair six-sided die follows a uniform distribution. The probability of any specific outcome is:

P(X = 4) = 1 / 6

Expectation (Expected Value)

The expectation, or expected value (E[X]), is a fundamental concept in probability theory. It represents the long-term average or mean value of a random variable. It gives you a sense of the "center" of a probability distribution.

1. Discrete Random Variable

If X is a discrete random variable, its expected value is calculated as:

E[X] = Σ x * P(X = x)

Where:

  • Σ denotes the sum over all possible outcomes of X,
  • P(X = x) is the probability of outcome x.

Example: Fair Die Roll

For a fair die roll, the expected value is:

E[X] = Σ (x * 1/6) = (1 + 2 + 3 + 4 + 5 + 6) / 6 = 3.5

This means that, on average, you expect to roll a 3.5 (which is not possible, but it represents the center of the distribution).
